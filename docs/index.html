HaeronClawHaeronClaw

Hosted agent runtime on Azure Functions. 基于 Azure Functions 托管的 Agent Runtime。

HaeronClaw is a branded, cloud-hosted markdown agent built for Microsoft and Azure workflows. It keeps the simple authoring model of AGENTS.md, skills, MCP servers, and Python tools, then adds Azure hosting, Teams delivery, enterprise integrations, and artifact handling so the same agent can run beyond local Copilot Chat. HaeronClaw 面向 Microsoft 与 Azure 工作流，保留 AGENTS.md、skills、MCP servers 和 Python tools 的创作方式，同时补上 Azure 托管、Teams 交付、文档制品生成、SharePoint/OneDrive 接入与企业集成能力，让同一个 agent 不仅能在本地 Copilot Chat 中运行，也能在云端作为服务运行。

Quick Install快速安装

Clone the repository and open src in VS Code to start chatting with the agent locally: 克隆仓库，在 VS Code 中打开 src 目录即可在本地与 agent 对话：

bash 
git clone https://github.com/haxudev/haeronclaw.git
cd haeronclaw

Deploy to Azure Functions on ACA with one command: 一行命令部署到 Azure Functions on ACA：

powershell 
./scripts/deploy.ps1 -Mode aca -ResourceGroup <rg-name> -Location eastus2 -Prefix fmaaca -Model github:gpt-5.4 -ImageTag v3

Tip: Python tools in src/tools/ only run in the cloud runtime. MCP servers and skills work locally in VS Code. 提示：src/tools/ 下的 Python 工具仅在云端运行时执行。MCP servers 和 skills 在本地 VS Code 中可用。

Core Capabilities核心能力

Cloud-Hosted Runtime云托管运行时

Deploy to Azure Functions on ACA with HTTP chat API, streaming SSE, and built-in chat UI.部署到 Azure Functions on ACA，提供 HTTP 聊天 API、流式 SSE 和内置聊天 UI。
🔌

Remote MCP Endpoint远程 MCP 接入

Built-in /runtime/webhooks/mcp for remote MCP clients to connect directly.内置 /runtime/webhooks/mcp 端点，让远程 MCP 客户端直连。
💬

Teams IntegrationTeams 集成

Async execution with proactive replies, typing indicators, and heartbeat progress.异步执行，支持主动回复、typing indicator 和心跳进度。
📦

Artifact Delivery制品交付

Generate PPTX, DOCX, XLSX, PDF and deliver via Blob Storage download links.生成 PPTX、DOCX、XLSX、PDF 并通过 Blob Storage 下载链接分发。
📧

M365 WorkflowsM365 工作流

Send email, read mail, browse OneDrive, query SharePoint, and transcribe voice.发送邮件、读取邮件、浏览 OneDrive、查询 SharePoint、语音转录。
💾

Session Persistence会话持久化

Multi-turn state persisted to Azure Storage for session resumption across restarts.多轮状态持久化到 Azure 存储，支持跨重启的会话恢复。
